Code for Japan

https://www.code4japan.org/

  • A civic tech network in Japan.
  • Helps people in Japan build their own communities, called brigade.
  • ex. Code for Ibaraki, Code for Chiba, Code for Okinawa, etc.
  • Brigades are totally independent.
  • Some people: Hal Seki, Shigeomi Shibata.

Code for Japan

  • CfJ helps gov in: capacitiy building, consulting, workshops, fellowships.
  • CfJ helps citizen in: community building, networking, project facilitation.
  • CfJ helps break walls between gov, citizen, and corporations.
  • "Start from the people, not from tech."

Code for Japan Summit

https://summit2019.code4japan.org/

  • Annual gathering of the CfJ communities, started at 2014.
  • One brigade holds the event starting from 2016.(?)
  • Usually ~1000 attendees.
  • Code for Chiba this year; Code for Aichi next year.

General impressions

  • Gov, citizen, corporation
  • Many public servants and researchers
  • Local field lab
  • Data academy

South Korean civic tech session

Civic tech movement also started around 2011-2012 in South Korea.

CodeNamu

http://codenamu.org/

  • The first civic hacking community in South Korea.
  • Founded around 2012. Inactive now.
  • “We Make Open Government”
  • Worked on translations, open API, data visualization, open data.

CodeNamu

  • Held the 1st hackathon in 2012-07-20.
  • “Let’s Share”
  • Mayor of Seoul city government attended.
  • In Ohyeon’s memory: “it was very tiring.”

PeaceCode

http://peacecode.org/

  • Founded around 2012. Inactive now.
  • “Writing code and making services for world peace”(?)
  • More freewheeling spirit. Some software were in beer-ware license.

PeaceCode

Their works:

  • ansim.me, 2012
    • Collect antibiotic usage data of hospitals.
    • “Ansim” means “feel safe”
    • Mapped out many open data sets for the safety of citizens.
    • Several years later government use the data to create similar service.

PeaceCode

Their works:

  • VoteAwards, 2012
    • Make voting more fun with crowd awarding platform.
    • Election is very important so it show be more fun.

Code for Seoul Community

http://codeforseoul.org/

  • An open community where anyone can participate to improve the city.
  • Founded around 2014. Inactive now.
  • A project by CodeNamu and Code for America.(?)
  • Hosts hack nights “Night Knight” every Monday.
  • Some people: Hoony.

Code for Seoul Community
